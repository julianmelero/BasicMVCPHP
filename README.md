# MVC Framework PHP
This a basic framework MVC in PHP, so please, **don't use it in production**.

# Requeriments
- Docker or PHP or LAMP
- Composer

# How it works
Using **Docker**:

- <code>docker-compose up -d php-apache</code>
- <code>composer dump</code>
- Go to <a href="localhost:8000">localhost:8000</a>
